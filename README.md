# Internship CRM

A full-stack Internship Management System designed to automate and simplify the internship process for students and administrators.

## Features

* Student Registration & Management
* Internship Application Tracking
* Offer Letter Generation
* Certificate Generation
* Admin Dashboard
* QR Verified Certificates
* Internship Status Tracking
* Document Management
* Secure Authentication & Authorization

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB

## Installation

### Clone Repository

```bash
git clone https://github.com/infocodrexa/Internship-CRM.git
```

### Install Frontend Dependencies

```bash
cd client
npm install
npm start
```

### Install Backend Dependencies

```bash
cd server
npm install
node index.js
```
